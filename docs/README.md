## 使用说明

### Claude Code 开发时

1. **先阅读设计文档**
   ```bash
   # 查看文档列表
   ls docs/
   
   # 开发前参考设计方案
   cat docs/README.md
   ```

2. **严格遵循文档规范**
   - DBF 字段格式参考 `迅投 PB-DBF 预埋单参数说明文档 V2.15.xlsx`
   - T0 逻辑参考 `手动 T0 实施方案.docx`

3. **有疑问先查文档**
   - 不要假设字段格式
   - 不要猜测业务逻辑
